Building and Running Minisip

IMPORTANT: This set of instruction is only intended for a specific version of minisip project.

Dependencies

openssl
libssl-dev
libglademm version >= 2.4
libgtkmm-2.4-dev
libltdl3-dev
build-essentials, g++, make, automake, autoconfig, libtool

Automated Build

Modified scripts available to be executed at minisip root repository directory: build all, clean all

Manual Build

IMPORTANT: Must be built in this order.
Compiling and installing libmutil First we run the bootstrap script to generate the configure script.
jeff@home:~/minisip/libmutil$ ./bootstrap
jeff@home:~/minisip/libmutil$ ./configure

Now we are ready to compile the source code of libmutil and install it:
jeff@home:~/minisip/libmutil$ make
jeff@home:~/minisip/libmutil$ make install


Then execute this echo /usr/local/share/aclocal >> /usr/share/aclocal/dirlist
Compiling and installing libmnetutil jeff@home:~/minisip/libmnetutil$ ./bootstrap
jeff@home:~/minisip/libmnetutil$ ./configure --enable-sctp
jeff@home:~/minisip/libmnetutil$ make
jeff@home:~/minisip/libmnetutil$ make install

NOTE: the --enable-sctp is to overcome a bug in /libmnetutil/m4/libmnetutil.m4
Compiling and installing
libmcrypto
libmikey
libmsip
libmstun
libminisip
(In that order) 		Use the same commands as before in its respective folders e.g. for libmcrypto use:
jeff@home:~/minisip/libmcrypto$ ./bootstrap
jeff@home:~/minisip/libmcrypto$ ./configure
jeff@home:~/minisip/libmcrypto$ make
jeff@home:~/minisip/libmcrypto$ make install
Compiling and installing minisip jeff@home:~/minisip/minisip$ ./bootstrap

We need to add configuration flags to configure minisip application
--enable-textui, --enable-gtk or --enable-qt

This sets which user interface will be used. GTK is the default one, and the text interface is often used during development. The QT one is not maintained any more and will probably not compile.

To enable textui:
jeff@home:~/minisip/minisip$ ./configure --enable-textui
jeff@home:~/minisip/minisip$ make
jeff@home:~/minisip/minisip$ make install

Running minisip

jeff@home:~/minisip/minisip$ minisip_gtkgui
Or
jeff@home:~/minisip/minisip$ minisip_textui
